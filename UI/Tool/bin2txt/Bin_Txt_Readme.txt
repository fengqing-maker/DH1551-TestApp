BinToTxt.exe使用说明：
命令格式：
    BinToTxt.exe  file1   file2
参数说明：
    file1: 二进制格式的文件
    file2: Assic数组格式的文件
例如：
    file1内容：12345678

    file2内容：{0x12, 0x34, 0x56, 0x78};
    

TxtToBin.exe使用说明：
命令格式：
    TxtToBin.exe  file1   file2
参数说明：
    file1: Assic数组格式的文件
    file2: 二进制格式的文件
例如：
    file1内容：{0x12, 0x34, 0x56, 0x78};
    file2内容：12345678
